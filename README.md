# Kantin Cerdas - Smart Canteen Management System
This repository contains the source code for **Smart Canteen**, a web-based application designed to streamline the ordering process and manage customer queues efficiently. This project was developed as a Capstone Project to address real-world queuing and order management issues at the canteens located in front of the Jember University Library (specifically Kantin Soto Banyuwangi and Kantin Perpustakaan).

## About the Project
Traditional order handling in crowded canteens often leads to unorganized queues, uncertain waiting times, and a poor customer experience. When canteens get busy, tenants struggle to determine which orders to process first, and customers are left in the dark about the status of their food.

**Smart Canteen** solves this problem by digitalizing the order workflow. It provides a seamless platform for customers to place and track their orders while empowering canteen tenants with a dedicated dashboard to manage incoming orders, update menus, and recapitulate sales data. 

## Key Features
The application is divided into two main user roles, each with tailored functionalities:

### For Customers
- **Product Ordering:** Easily browse the available food and beverage menus from different canteen tenants and place orders digitally without standing in a long physical queue.
- **Order Tracking:** Provides real-time visibility into the order status. Customers can see exactly when their order is being prepared and when it is ready to be picked up.

### For Tenants (Canteen Owners)
- **Order Management:** A structured dashboard to receive, organize, and update the status of incoming customer orders, ensuring a first-in, first-out workflow.
- **Menu Management:** Allows tenants to easily add, update, or remove food and beverage items from their digital menu.
- **Order Data Recapitulation:** Automatically records and summarizes order history to help tenants review their sales data efficiently.


## How to Use the Application
### As a Customer:
1. Open the Kantin Cerdas website in your browser.
2. Select the specific canteen or tenant you want to order from.
3. Browse the menu and select your desired food or beverages.
4. Submit your order and receive an order tracking status.
5. Monitor your order status and pick up your meal once the status changes to ready.

### As a Tenant:
1. Log in to the tenant dashboard using your credentials.
2. Navigate to the **Order Management** section to view real-time incoming orders.
3. Update the status of each order (e.g., Processing, Ready, Completed) as you prepare the food.
4. Use the **Menu Management** tab to update product availability or add new items.
5. Check the **Recapitulation** section to review past order records.
